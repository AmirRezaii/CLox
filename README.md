# **Clox** - Programming Language
Clox is a turing complete programming language written in C.
## Examples
Some examples are located in the tests directory including:
- clock.lox
- closure.lox
- fibonacci.lox
- sum.lox
## Building
Clone this git repository and cd to it.
``` sh
git clone https://github.com/AmirRezaii/CLox
cd CLox/
```
Build the binary using NOB with the command below.
``` sh
gcc nob.c && ./nob
```
CLox binary is located now in the build directory.
